Variables LET

En Javascript usamos let para crear variables cuyo valor puede cambiar durante la ejecución del programa

Ejemplo:

let nombre = "Rubén";

Tipos de datos
  • String: Texto o cadena de caracteres -> "Hola que áse"
  • Número Entero: Número de tipo entero -> 23573
  • Número Decimal: Número de tipo decimal -> 3,1416
  • Booleno: Dos valores posibles, verdadero o falso -> true
Operadores
  • Suma: + -> a+b
  • Resta: - -> a-b
  • Producto o Multiplicación: * -> a*b
  • División: / -> a/b
  • Resto o Módulo % -> a%
